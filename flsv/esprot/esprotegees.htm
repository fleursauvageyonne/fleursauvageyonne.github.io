Parmi les plantes actuellement disparues dans l'Yonne, une seule semble donc éteinte :

  • Une Violaceae :
    - Viola cryana

  • Deux Orchidaceae :
    - Anacamptis coriophora

  • Une Apiaceae :
    - Helosciadium repens

  • Plusieurs Asteraceae :

    - Pulicaria vulgaris

  • Fabaceae :

  • Une Hypericaceae :
    - Hypericum elodes

  • Menyanthaceae :

  • Orobanchaceae :

  • Paeoniaceae :
    - Paeonia mascula

  • Papaveraceae :

  • Ranunculaceae :
